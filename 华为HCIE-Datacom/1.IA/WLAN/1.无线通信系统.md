```
无线通信系统：  
无线通信系统中，信息可以是图像、文字、声音等。  
信息需要先经过信源编码转换为方便于电路计算和处理的数字信号，再经过信道编码和调制，转换为无线电波发射出去。
```

```
WLAN技术就是通过无线电磁波在空间中传输信息。  
当前我们使用的频段是：  
2.4GHz频段 (2.4GHz~2.4835GHz）；  
5GHz频段（5.15GHz~5.35GHz，5.725GHz~5.85GHz）。
```
 
```
特高频 (300MHz–3GHz) ：  
电视广播、无线电话通讯、无线网络、微波炉。  
超高频 (3GHz–30GHz) ：  
无线网络、雷达、人造卫星接收。
```
