```
什么是IP地址？  
网络设备的标识，每一个网络设备都需要存在IP地址  
IP地址相当于现实中的地址
```
 
```
IP地址的使用：  
在网络设备的网卡中需要获取到IP地址，设备才能上网通信
```
 
```
IP地址类型：  
1.网络地址：描述该地址网络的范围  
    通过二进制换算后，将网络部分不变，主机部分全部变为0  
    192.168.10.0
```
 
```
2.主机地址：这个网络范围内被设备使用的地址  
    网络地址+1  到 广播地址-1  
    192.168.10.1 - 192.168.10.254
```
 
```
3.广播地址：这个网络范围的广播报文  
    通过二进制换算后，将网络部分不变，主机部分全部变为1  
    192.168.10.255
```
 
```
IP地址的表示：  
1.IP地址长度是4字节，32bit，地址数据为2^32  
2.表示方法：  
    1.二进制表示  
        32个 1 或 0 表示  
    2.点分十进制表示  
        通过4个十进制数 中间点号 分隔
```
 
```
二进制和十进制的转换：  
1.二进制转换为十进制  
  将8个bit 从左到右 依次对于数列值 从大到小  
  将bit值为1 对于的数列值 进行相加  
  相加得到的和值就是十进制值  
    
  *一个十进制数 最小值为 0  
  *一个十进制数 最大值为 255  
   所以一个十进制数的取值范围是 0 - 255
```
 
```
2.十进制转换为二进制  
  将十进制 从大到小 依次减去数列值  
  如果减去后 得到的差值 大于等于0，则该位bit置1  
  如果减去后 得到的差值 小于0，则该位bit置0  
  如果差值小于0，则保留数值继续下一个数列值的减运算
```
 
```
IP地址的取值范围：  
  0.0.0.0 - 255.255.255.255
```
 
```
IP地址分类：  
1.A类地址，第一个bit固定为0，0.0.0.0 - 127.255.255.255  
     默认网络掩码为 255.0.0.0 （ /8 ）  
2.B类地址，前两个bit固定为10，128.0.0.0 -   
191.255.255.255  
     默认网络掩码为 255.255.0.0 （ /16 ）  
3.C类地址，前三个bit固定为110，192.0.0.0 -   
223.255.255.255  
     默认网络掩码为 255.255.255.0 （ /24 ）  
4.D类地址，前四个bit固定为1110，224.0.0.0 -   
239.255.255.255  
5.E类地址，前四个bit固定为1111，240.0.0.0 -   
255.255.255.255
```
 
```
正常网络设备使用的单播地址是 A类、B类、C类  
D类地址是组播地址  
E类地址是研究测试使用的
```
 
```
A类、B类、C类地址进一步划分（二次分类）：  
1.公网地址  
   全网唯一的  
2.私网地址  
   可以重复利用的  
   A类地址： 10.0.0.0 - 10.255.255.255  
   B类地址： 172.16.0.0 - 172.31.255.255  
   C类地址： 192.168.0.0 - 192.168.255.255
```
 
```
特殊IP地址：  
1.  0.0.0.0          设备没有地址，需要得到一个地址  
2.  255.255.255.255   全网广播，在任意一个网段内，表示广  
播报文  
3.  127.0.0.0-127.255.255.255    测试地址（环回地址），  
用于检查设备是否存在报文处理协议故障等问题  
4.  169.254.0.0-169.254.255.255  链路地址，在没有获取地  
址后，会使用该地址做链路通信
```

子网划分
```
    
```
为什么要进行子网划分？  
 一个C类地址可以存在多少个可用的IP？  254  
 一个B类地址可以存在多少个可用的IP？  65534  
   
一间教室存在30台电脑上网，分配一个C类地址，  
但是会有大量的地址剩余，造成地址资源的浪费  
以此需要使用子网划分，进行地址资源的合理分配
```
 
```
子网划分还可以将地址进行有规律、有条例的进行分配
```
   

```
一间教室存在100台电脑上网，给管理员分配地址192.168.10.123/24  
问如何进行子网划分最合理？  
现有网段 192.168.10.0/24   （网络地址 + 网络掩码 = 网段）
```
 
```
    2^n - 2 \>=100   （n取最小值）  
 n   =   7  （主机位就是主机部分的bit个数）  
      32 - 7 =  25  （网络位就是网络部分的bit个数）
```
 
```
所以可以为教室分配的网段为 192.168.10.0/25  
             网络地址：192.168.10.0  
 可用地址：192.168.10.1-126  
 广播地址：192.168.10.127  
剩余网段为  192.168.10.128/25  
    
IP地址   192.168.10.0    1100 0000  1010 1000  0000 1010  1000 0000  
网络掩码 255.255.255.0   1111 1111  1111 1111  1111 1111  1100 0000
```

VLSM：可变长的子网掩码
```

```
IP地址的构成：  
  192.168.10.1  255.255.255.0（/24）  
1.网络部分：描述了一个大的范围  
2.主机部分：在范围内单独的一个设备  
如果区分网络部分和主机部分？  
  需要通过网络掩码进行区分  
网络掩码的表示：  
  1.二进制表示  
  2.点分十进制表示
```
 
```
网络掩码的特点：  
  1.从二进制看，必须是连续的1和连续的0组合  
  2.bit 1 在bit 0之前  
网络掩码的作用：  
  1.从二进制看，网络掩码 bit为1 对于IP地址的 就是网络部分  
  2.从二进制看，网络掩码 bit为0 对于IP地址的 就是主机部分
```
