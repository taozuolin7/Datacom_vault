```
LSPID：链路状态信息的描述  
        0000.0000.0001       .00                    -00                  *  
         system-id       伪节点标识符           分片标识符           自身的LSP标识  
                     如果为全0表示实节点    如果为全0表示非分片  
                     如果不为全0表示伪节点  如果不为全0表示分片  
seq：序列号  
chksum：校验和  
holdtime：保持时间  
           取值时间是 0-1200s ，从1200s开始进行递减 ，900s后更新  
length：长度，LSP内容的长度表示  
ATT/P/OL: 功能bit
```

    
```
[AR3]dis isis lsdb 0000.0000.0003.00-00 verbose level-1  
对于LSP的唯一判断：根据LSPID值进行判断  
对于LSP的新旧判断：seq、chksum、holdtime  
先比较seq，seq值越大越优先  
seq相等，比较age，判断age是否等于0：  
       如果为0（表示撤销），则优选  
       如果不为0，则认为age相同  
age相等，比较chksum，优选校验和大的值
```

```
影响ISIS邻居建立的因素：  
1. L1的邻居必须配置相同的区域ID  
2. 邻居之间的systemid要求不能一致  
   如果是在同级别区域内出现了两台设备systemid冲突，则会导致邻居无法正常建立  
   如果是在不同级别区域内出现了两台设备systemid冲突，则邻居正常建立  
3. 邻居之间的网段可以不相同，但要求对端的IP地址在本端的网段范围内
```
