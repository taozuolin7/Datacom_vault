**场景概述：**
1.底层通过 static-route 互访，通过loopback 0 建立隧道
2.AR1作为隧道入口

**查询命令：**
```R
display mpls static-cr-lsp verbose
display mpls interface
```

**关键配置：**
AR1
```C
mpls lsr-id 1.1.1.1
mpls
 mpls te
 
#
interface GigabitEthernet0/0/0
 ip address 10.1.12.1 255.255.255.0 
 mpls
 mpls te
 mpls te bandwidth max-reservable-bandwidth 200000 #静态 CR-LSP 配置了 ct0，那么就需要再接口上配置
 mpls te bandwidth bc0 100000
#

interface Tunnel0/0/0
 ip address unnumbered interface LoopBack0
 tunnel-protocol mpls te
 destination 3.3.3.3
 mpls te signal-protocol cr-static
 mpls te tunnel-id 1
 mpls te commit
 mpls
 
static-cr-lsp ingress tunnel-interface Tunnel0/0/0 destination 3.3.3.3 nexthop 10.1.12.2 out-label 102 bandwidth ct0 100000 # tunnel-interface 不是 自定义名称，需要注意
static-cr-lsp egress AR3-AR1 incoming-interface GigabitEthernet0/0/0 in-label 201 lsrid 3.3.3.3 tunnel-id 2
```

AR2:
```C
interface GigabitEthernet0/0/0
 ip address 10.1.12.2 255.255.255.0 
 mpls
 mpls te
 mpls te bandwidth max-reservable-bandwidth 200000
 mpls te bandwidth bc0 100000
#
static-cr-lsp transit AR1-AR3 incoming-interface GigabitEthernet0/0/0 in-label 102 nexthop 10.1.23.3 out-label 203 bandwidth ct0 100000

```

AR3:
```C
interface GigabitEthernet0/0/0
 ip address 10.1.23.3 255.255.255.0 
 mpls
 mpls te
 mpls te bandwidth max-reservable-bandwidth 200000
 mpls te bandwidth bc0 100000
 
 static-cr-lsp egress AR1-AR3 incoming-interface GigabitEthernet0/0/0 in-label 203 lsrid 1.1.1.1 tunnel-id 1
```
